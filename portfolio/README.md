# Portfolio

A simple, responsive personal portfolio site. Built with vanilla HTML/CSS/JS.

## Preview locally

```bash
python3 -m http.server 8000 --directory /workspace/portfolio
```
Then open http://localhost:8000 in your browser.

## Customize
- Replace "Your Name" in `index.html`.
- Update social links and email.
- Swap project cards with your own content and links.
- If you use the contact form, replace the `action` with your Formspree or backend endpoint.

## Deploy
- GitHub Pages: push `/workspace/portfolio` contents to a `gh-pages` branch or set as Pages source.
- Netlify/Vercel: import the repo and set the publish directory to `/workspace/portfolio`.
